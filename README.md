Swift版本的iOS开发框架，主要对第三方库进行封装，集成CI、Test框架。

## 三方依赖库
目前采用cocoapods管理第三方库，尝试使用了smp进行包管理，感觉目前还是不成熟T^T。

[三方依赖库列表](https://github.com/yeziahehe/YFSwiftFramework/blob/master/Dependence.md)
